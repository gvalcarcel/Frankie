# Servidores

## Proposito

Inventariar servidores, VMs y hosts gestionados por Frankie.

## Estructura sugerida

```markdown
## Nombre del servidor

- Rol:
- Sistema operativo:
- IP o nombre:
- Entorno:
- Servicios principales:
- Responsable:
- Estado:
```

## Ejemplo

```markdown
## ejemplo-servidor

- Rol: laboratorio
- Sistema operativo: Linux
- Servicios principales: pendiente
```
