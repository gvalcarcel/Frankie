# Aula Frankie

## Para qué sirve esta carpeta

Aquí encontrarás materiales para aprender administración de sistemas con Frankie Core.

Las prácticas son **offline**. Leen información guardada en el repositorio. No se conectan al servidor físico Frankie ni cambian sus máquinas virtuales.

## Ideas importantes

- Frankie es el servidor físico del aula.
- Frankie Core es el programa que consulta el conocimiento del repositorio.
- `OK` no significa “perfecto para siempre”. Significa que la evidencia disponible no muestra un problema activo.
- Un aviso se estudia antes de hacer cambios.
- Live Mode y Repair Mode no están implementados.
- Una tarea LIVE solo podrá empezar con autorización expresa y una guía concreta.

## Para el profesorado

Las ocho actividades incluyen objetivo, material, pasos, resultado esperado, preguntas y mini rúbrica. Pueden realizarse de forma individual o por parejas. No necesitan acceso a la red del aula.
